# Rules of JSX?

1. self closing or paired tag
2. Adjacent JSX elements must be <></> React fragment sugar form
3. class ---> className
4. kebab-case ---> camelCase
   background-color (kebab case) ---> bakgroundColor camelCase
5. Javascript in jsx {}
